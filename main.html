Question-1: What are the different ways to select an element in the DOM?

Different ways to select elements in the DOM include methods like getElementById(), querySelector(), and getElementsByClassName(), querySelectorAll() ,firstChild/last child each tailored for specific selection needs.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML includes HTML tags, innerText returns visible text with styles applied, and textContent returns all text, including hidden, without considering styles, offering better performance than innerText.

Question-3: What is event delegation in the DOM?

Event Delegation involves adding a single event listener to a parent element instead of individual child elements. The event.target property helps identify the specific element that triggered the event.

Question-4: What is event bubbling in the DOM?

Event bubbling in JavaScript is a process where an event triggered on a child element propagates upward to its parent elements. This allows parent elements to handle events triggered by child elements.
Event propagation in JavaScript occurs in two phases:

  • Capturing Phase: The event starts from the root and propagates down to the target element.
  • Bubbling Phase: After reaching the target, the event bubbles up from the target to the root.

Question-5: How do you create, add, and remove elements using JavaScript?

We can create elements with document.createElement(), add them using appendChild() or insertBefore(), and remove them with removeChild() or remove().
